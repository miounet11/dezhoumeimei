# PokerIQ Pro 完整功能清单

## 📊 系统架构概览

### 技术栈
- **前端框架**: Next.js 15.4.6 (App Router)
- **UI组件库**: Ant Design 5.x
- **样式方案**: Tailwind CSS + CSS Modules
- **状态管理**: Redux Toolkit + localStorage
- **数据库**: Prisma ORM + SQLite/PostgreSQL
- **认证方案**: NextAuth (配置中) + JWT (临时)
- **数据可视化**: Recharts
- **构建工具**: Turbopack
- **语言**: TypeScript 5.x
